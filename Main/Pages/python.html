Python Variables

In Python, variables are used to store data values. A variable is essentially a name that refers to a value. Python uses dynamic typing, which means that you don’t need to declare the type of a variable explicitly; the type is inferred from the value assigned to it.

Creating Variables

To create a variable, simply assign a value to a name using the = operator. Python allows you to use a wide range of variable names, provided they follow the naming rules.

  • Variable Naming Rules: Variable names must start with a letter or an underscore (_), followed by letters, digits, or underscores. Variable names are case-sensitive.
  • Examples of Valid Variable Names: my_var, value1, _temp
  • Examples of Invalid Variable Names: 1value, my-var, value@

Example: Basic Variable Usage

Here’s a simple example demonstrating how to create and use variables:

        
        # Assigning values to variables
        name = "Alice"
        age = 30
        
        # Using variables
        print("Name:", name)  # Output: Name: Alice
        print("Age:", age)    # Output: Age: 30

Variable Types

Python supports several data types that can be assigned to variables, including:

  • Integers: Whole numbers, e.g., x = 10
  • Floats: Decimal numbers, e.g., y = 3.14
  • Strings: Text values enclosed in quotes, e.g., name = "Alice"
  • Booleans: True or False values, e.g., is_active = True
  • Lists: Ordered collections of items, e.g., numbers = [1, 2, 3]

Example: Variable Types and Operations

This example demonstrates different types of variables and basic operations:

        
        # Integer and Float
        x = 10
        y = 3.14
        result = x + y
        print("Result:", result)  # Output: Result: 13.14
        
        # String
        greeting = "Hello, "
        name = "Alice"
        message = greeting + name
        print("Message:", message)  # Output: Message: Hello, Alice
        
        # Boolean
        is_active = True
        print("Is active:", is_active)  # Output: Is active: True

Variable Scope

The scope of a variable refers to where the variable is accessible within the code. Variables defined inside a function are local to that function, while variables defined outside of any function are global.

  • Local Variables: Variables defined within a function and accessible only within that function.
  • Global Variables: Variables defined outside of all functions and accessible throughout the entire script.

Example: Variable Scope

Here’s an example showing local and global variables:

        
        # Global variable
        global_var = "I am global"
        
        def my_function():
            # Local variable
            local_var = "I am local"
            print(local_var)  # Output: I am local
            print(global_var) # Output: I am global
        
        my_function()
        print(global_var)  # Output: I am global
        # print(local_var)  # This will cause an error because local_var is not accessible here

Python Data Types

Python supports a variety of data types that allow you to work with different kinds of data. Each data type serves a specific purpose and is used to store and manipulate data in your programs. Understanding data types is crucial for effective programming in Python.

Basic Data Types

Python has several built-in data types that are commonly used:

  • Integers: Whole numbers without a decimal point. Examples include 5, -42, and 1000.
  • Floats: Numbers with a decimal point. Examples include 3.14, -0.001, and 2.0.
  • Strings: Sequences of characters enclosed in single quotes (') or double quotes ("). Examples include "Hello" and 'World'.
  • Booleans: Represents truth values True and False.

Example: Basic Data Types

Here’s a simple example demonstrating different data types:

        
        # Integer
        age = 25
        
        # Float
        height = 5.9
        
        # String
        name = "Alice"
        
        # Boolean
        is_student = True
        
        print("Age:", age)          # Output: Age: 25
        print("Height:", height)    # Output: Height: 5.9
        print("Name:", name)        # Output: Name: Alice
        print("Is student:", is_student)  # Output: Is student: True

Composite Data Types

In addition to basic data types, Python also supports composite data types that are used to store collections of values:

  • Lists: Ordered collections of items that can be of different types. Lists are defined using square brackets ([]). Example: [1, 2, 3, "Python"].
  • Tuples: Similar to lists, but immutable (cannot be changed after creation). Tuples are defined using parentheses (()). Example: (1, 2, 3, "Python").
  • Sets: Unordered collections of unique items. Sets are defined using curly braces ({}). Example: {1, 2, 3, "Python"}.
  • Dictionaries: Collections of key-value pairs. Dictionaries are defined using curly braces with key-value pairs separated by colons ({key: value}). Example: {'name': 'Alice', 'age': 25}.

Example: Composite Data Types

Here’s how you can use composite data types in Python:

        
        # List
        my_list = [1, 2, 3, "Python"]
        print("List:", my_list)  # Output: List: [1, 2, 3, 'Python']
        
        # Tuple
        my_tuple = (1, 2, 3, "Python")
        print("Tuple:", my_tuple)  # Output: Tuple: (1, 2, 3, 'Python')
        
        # Set
        my_set = {1, 2, 3, "Python"}
        print("Set:", my_set)  # Output: Set: {1, 2, 3, 'Python'}
        
        # Dictionary
        my_dict = {'name': 'Alice', 'age': 25}
        print("Dictionary:", my_dict)  # Output: Dictionary: {'name': 'Alice', 'age': 25}

Type Conversion

Python allows you to convert between different data types using built-in functions. This is useful when you need to perform operations that require specific data types:

  • Convert to Integer: int()
  • Convert to Float: float()
  • Convert to String: str()
  • Convert to List: list()
  • Convert to Tuple: tuple()

Example: Type Conversion

This example demonstrates how to convert between different data types:

        
        # Convert to Integer
        x = "10"
        x_int = int(x)
        print("Integer:", x_int)  # Output: Integer: 10
        
        # Convert to Float
        y = "3.14"
        y_float = float(y)
        print("Float:", y_float)  # Output: Float: 3.14
        
        # Convert to String
        z = 25
        z_str = str(z)
        print("String:", z_str)  # Output: String: 25

Python Operators

Operators in Python are special symbols or keywords used to perform operations on values and variables. They are used in various expressions and statements to manipulate data. Python supports several types of operators, each serving a different purpose.

Types of Operators

Python includes the following types of operators:

1. Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations:

  • Addition: + (e.g., 5 + 3)
  • Subtraction: - (e.g., 5 - 3)
  • Multiplication: * (e.g., 5 * 3)
  • Division: / (e.g., 5 / 3)
  • Modulus: % (e.g., 5 % 3)
  • Exponentiation: ** (e.g., 5 ** 3)
  • Floor Division: // (e.g., 5 // 3)

Example: Arithmetic Operators

Here’s an example demonstrating the use of arithmetic operators:

        
        # Arithmetic Operations
        a = 10
        b = 3
        
        addition = a + b
        subtraction = a - b
        multiplication = a * b
        division = a / b
        modulus = a % b
        exponentiation = a ** b
        floor_division = a // b
        
        print("Addition:", addition)           # Output: Addition: 13
        print("Subtraction:", subtraction)     # Output: Subtraction: 7
        print("Multiplication:", multiplication) # Output: Multiplication: 30
        print("Division:", division)           # Output: Division: 3.3333333333333335
        print("Modulus:", modulus)             # Output: Modulus: 1
        print("Exponentiation:", exponentiation) # Output: Exponentiation: 1000
        print("Floor Division:", floor_division) # Output: Floor Division: 3

2. Comparison Operators

Comparison operators are used to compare values:

  • Equal to: == (e.g., 5 == 3)
  • Not equal to: != (e.g., 5 != 3)
  • Greater than: > (e.g., 5 > 3)
  • Less than: < (e.g., 5 < 3)
  • Greater than or equal to: >= (e.g., 5 >= 3)
  • Less than or equal to: <= (e.g., 5 <= 3)

Example: Comparison Operators

Here’s an example demonstrating the use of comparison operators:

        
        # Comparison Operations
        x = 10
        y = 20
        
        print("x == y:", x == y)   # Output: x == y: False
        print("x != y:", x != y)   # Output: x != y: True
        print("x > y:", x > y)  # Output: x > y: False
        print("x < y:", x < y)  # Output: x < y: True
        print("x >= y:", x >= y) # Output: x >= y: False
        print("x <= y:", x <= y) # Output: x <= y: True

3. Logical Operators

Logical operators are used to combine conditional statements:

  • AND: and (e.g., (x > 5) and (x < 10))
  • OR: or (e.g., (x > 5) or (x < 10))
  • NOT: not (e.g., not(x > 5))

Example: Logical Operators

Here’s an example demonstrating the use of logical operators:

        
        # Logical Operations
        a = 10
        b = 20
        
        print("a > 5 and b < 30:", a > 5 and b < 30)  # Output: a > 5 and b < 30: True
        print("a < 5 or b > 15:", a < 5 or b > 15)  # Output: a < 5 or b > 15: True
        print("not(a > 5):", not(a > 5))  # Output: not(a > 5): False

4. Assignment Operators

Assignment operators are used to assign values to variables:

  • Assign: = (e.g., x = 10)
  • Add and assign: += (e.g., x += 5)
  • Subtract and assign: -= (e.g., x -= 5)
  • Multiply and assign: *= (e.g., x *= 5)
  • Divide and assign: /= (e.g., x /= 5)
  • Modulus and assign: %= (e.g., x %= 5)

Example: Assignment Operators

Here’s an example demonstrating the use of assignment operators:

        
        # Assignment Operations
        x = 10
        
        x += 5  # Equivalent to x = x + 5
        print("x after += 5:", x)  # Output: x after += 5: 15
        
        x -= 3  # Equivalent to x = x - 3
        print("x after -= 3:", x)  # Output: x after -= 3: 12
        
        x *= 2  # Equivalent to x = x * 2
        print("x after *= 2:", x)  # Output: x after *= 2: 24
        
        x /= 4  # Equivalent to x = x / 4
        print("x after /= 4:", x)  # Output: x after /= 4: 6.0

Python Functions

Functions are a fundamental concept in Python programming. They allow you to group a set of statements together so they can be run more than once. Functions help make code more modular, reusable, and easier to understand.

Defining a Function

Functions in Python are defined using the def keyword, followed by the function name and parentheses. The function body is indented and contains the statements to be executed when the function is called:

            
            def greet(name):
                """Display a greeting message."""
                print("Hello, " + name + "!")

Calling a Function

To execute the code inside a function, you need to call it by using its name followed by parentheses. You can pass arguments to the function if it requires them:

            
            # Call the function with an argument
            greet("Alice")
            # Output: Hello, Alice!

Function Parameters and Arguments

Functions can take parameters, which are variables listed inside the parentheses in the function definition. When you call the function, you pass values to these parameters, known as arguments:

  • Positional Arguments: Arguments that are passed to the function in the order that the parameters are defined.
  • Keyword Arguments: Arguments that are passed by explicitly specifying the parameter name.
  • Default Arguments: Parameters that assume a default value if a value is not provided in the function call.

Example: Function Parameters

This example demonstrates different types of function parameters:

            
            def describe_pet(name, animal_type="dog"):
                """Display information about a pet."""
                print("I have a " + animal_type + " named " + name + ".")
            
            # Positional Argument
            describe_pet("Buddy")  # Output: I have a dog named Buddy.
            
            # Keyword Argument
            describe_pet(name="Whiskers", animal_type="cat")  # Output: I have a cat named Whiskers.
            
            # Default Argument
            describe_pet("Charlie")  # Output: I have a dog named Charlie.

Returning Values

Functions can return values using the return statement. The return value can be used in expressions or assigned to variables:

            
            def add_numbers(x, y):
                """Return the sum of two numbers."""
                return x + y
            
            result = add_numbers(5, 3)
            print("The sum is:", result)  # Output: The sum is: 8

Lambda Functions

Lambda functions are small anonymous functions defined using the lambda keyword. They can have any number of arguments but only one expression. The expression is evaluated and returned:

            
            # Lambda Function Example
            add = lambda x, y: x + y
            print("The sum is:", add(5, 3))  # Output: The sum is: 8

Python Modules

Modules are a way to organize and reuse code in Python. A module is a file containing Python code, which can define functions, classes, and variables. Modules help you break down large programs into smaller, manageable, and logically organized files.

Importing Modules

You can import modules using the import statement. This allows you to access the functions, classes, and variables defined in the module:

                
                # Importing the entire module
                import math
                
                # Using a function from the math module
                print(math.sqrt(16))  # Output: 4.0

Importing Specific Items

Instead of importing the entire module, you can import specific functions or variables. This can be done using the from keyword:

                
                # Importing a specific function
                from math import pi
                
                print(pi)  # Output: 3.141592653589793

Creating Your Own Modules

You can create your own modules by saving your Python code in a file with a .py extension. For example, if you have a file named my_module.py with the following code:

                
                # my_module.py
                def greet(name):
                    """Display a greeting message."""
                    print("Hello, " + name + "!")

You can import and use this module in another script as follows:

                
                # Importing the custom module
                import my_module
                
                # Using the function from the module
                my_module.greet("Alice")  # Output: Hello, Alice!

Module Aliases

To make your code more concise, you can use an alias when importing a module. This is done using the as keyword:

                
                # Importing a module with an alias
                import numpy as np
                
                # Using the alias to call a function from the numpy module
                array = np.array([1, 2, 3])
                print(array)  # Output: [1 2 3]

Standard Library Modules

Python comes with a rich standard library of modules. Some commonly used standard modules include os, sys, datetime, and json. These modules provide essential functionalities for various tasks:

                
                # Example using the os module
                import os
                
                # Get the current working directory
                print(os.getcwd())
